Template Installation

It’s easy to install Template . Just follow these steps, they won’t take much of your time.

Angular Setup -

Open terminal and run npm install -g @angular/cli

Angular CLI Doc: https://cli.angular.io

  1. Install NodeJs

    Your computer must have installed nodejs to run this template. You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js

  2. Install Yarn

    Your computer must have installed Yarn globaly to run this template. You can install yarn globaly by follow this url https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable

  3. Install Npm Packages

    You must have to install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.

  4. Development server

    Run yarn dev from the root of your project to start a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.


  5. Build

    Run yarn build to build the project. The build artifacts will be stored in the .next/ directory.


  6. Deploy

    Follow this link to learn about deployment

Root Folder Structure

  • angular.json

  • Contains all dependencies used for production and development.
  • src
  • Contains all Angular Typescript code, assets and basically everything the end user will have access to.
  • tsconfig.json
  • TypeScript compiler configuration.
